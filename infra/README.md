# Infrastructure

Deployment and local stack configuration for the 3PL visibility platform.

| Path | Purpose |
|------|---------|
| `../docker-compose.yml` | Root Compose file for API, frontend, database, and workers |
| `seed/` | Seed data scripts for local demos |
| `migrations/` | Database migration helpers |

Detailed service definitions and startup commands are documented in [docs/runbook.md](../docs/runbook.md).
